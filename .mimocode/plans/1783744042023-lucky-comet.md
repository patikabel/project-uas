# Plan: Sync Features + Hapus Aspirasi

## Goal
Connect chat, voting between warga↔admin via localStorage, and add delete aspirasi in admin.

## 1. Chat Sync (warga ↔ admin)
**Files:** `app/dashboard/page.tsx`, `app/admin/dashboard/page.tsx`

- On chat send: save messages to `localStorage("chat_data")` with room ID
- On mount: load chat history from localStorage
- Both dashboards read/write the same localStorage key
- Format: `{ roomId: number, sender: "user"|"petugas", name: string, text: string, time: string }[]`

## 2. Voting Sync (warga → admin)
**Files:** `app/dashboard/page.tsx`, `app/admin/dashboard/page.tsx`

- On warga vote: save vote to `localStorage("voting_data")` as `{ voted: boolean, candidateId: number }`
- On admin mount: load vote counts from localStorage, merge with initial `kandidatList`
- Admin sees updated suara counts from warga votes

## 3. Hapus Aspirasi in Admin
**File:** `app/admin/dashboard/page.tsx`

- Add delete button (IconTrash) in aspirasi detail modal and table row
- On delete: remove from `aspirasiList` state + remove from `localStorage("aspirasi_warga")`
- Confirmation dialog before delete

## Verification
- Login warga → send chat → login admin → see chat message
- Login warga → vote → login admin → see updated vote count
- Login admin → delete aspirasi → refresh → aspirasi gone
